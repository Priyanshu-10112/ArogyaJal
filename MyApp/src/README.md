# Project Structure

```
src/
├── screens/           # Screen components
│   ├── LaunchScreen.js    # Splash/Launch screen
│   └── HomeScreen.js      # Main home screen
├── components/        # Reusable components
├── navigation/        # Navigation configuration

│   └── images/       # Image files
├── constants/        # App constants

```
